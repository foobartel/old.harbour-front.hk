Talk 1 - Adam Morse

What is CSS for?

This talk will explore a simple question: 'What is CSS for?' It will examine the ways in which CSS can make a website better and the various ways it can make a website worse. By highlighting some core tenants of how HTML works, we'll explore some ways we can design with CSS to enhance a user's experience on the web.

Bio

Talk 2 - Edan Kwan

Creative Development with Animation

Edan Kwan will show an overview of advanced frontend/creative development. In this session, he will explain more about tween engine easing and he will round up with some live coding with GLSL (Using THREE.JS and/or PIXI.JS).
